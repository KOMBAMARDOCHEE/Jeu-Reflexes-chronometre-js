# Jeu de Réflexes Chronométré

## Présentation
Jeu de Réflexes Chronométré est une application web interactive permettant de 
mesurer le temps de réaction d’un utilisateur face à un stimulus visuel.  
Le projet met en avant la précision temporelle, la gestion des événements 
et une logique JavaScript claire et efficace.

## Objectifs techniques
- Mesurer un temps de réaction en millisecondes
- Gérer des délais aléatoires
- Manipuler le DOM de manière fiable
- Offrir une expérience utilisateur simple et immédiate

## Fonctionnalités
- Déclenchement aléatoire du signal
- Calcul précis du temps de réaction
- Affichage dynamique du résultat
- Réinitialisation automatique du jeu

## Architecture du projet
reaction-game/
 index.html
 style.css
 game.js


# Technologies utilisées
- HTML5
- CSS3
- JavaScript Vanilla (ES5 compatible navigateur)

## Installation et exécution
1. Télécharger ou cloner le projet
2. Ouvrir le fichier `index.html` dans un navigateur moderne
3. Cliquer lorsque le bloc devient vert

Aucun serveur n’est requis.
